# RnWechat
# 介绍
使用react native 初步尝试，做了一个知己APP，界面是仿微信页面，由于是Demo练习项目，只做了一部分，用户通信使用的是极光推送，并且只做了Android 兼容，iOS 不考虑。

# 软件架构
目前使用的React native 做的一个练习app,使用了一些 RN相关的库

# 安装教程

cd ./RnWechat
yarn install
yarn add android
